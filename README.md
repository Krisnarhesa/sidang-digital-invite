# 🎓 Digital Event Invitation & RSVP Platform

A modern, interactive, tech-themed digital invitation platform designed specifically for Thesis Defense (Sidang Skripsi) and Proposal Seminar events at Information Technology at Udayana University. Built with Next.js and Supabase, It features real time RSVP tracking, interactive UI elements, and a dedicated admin dashboard to manage guest attendance.

## Key Features

- **Tech & Luxury Aesthetics**: A unique design language featuring typing code animations, blinking cursors, an interactive particle node background, and an elegant color scheme (Mocha, Gold, Cream) that bridges the gap between technology and formal luxury.
- **Interactive "Degree Reveal"**: A gamified UI component where guests can click a black "censored tape" to interactively reveal the graduates' newly earned academic degrees, accompanied by celebratory confetti animations.
- **Real Time RSVP System**: A fully integrated RSVP form connected to a Supabase backend. Guests can seamlessly confirm their attendance, leave personalized wishes, and instantly receive a digital "Thank You" confirmation card.
- **Smart URL Parameters**: The platform supports dynamic URL routing (e.g., `?to=John+Doe`) to automatically personalize the invitation text and auto-fill the RSVP form for a frictionless guest experience.
- **Admin Dashboard**: A hidden dashboard route (`/admin`) providing real-time oversight of guest attendance, dynamically categorizing guests by event type and displaying their status and messages.
- **Immersive Background Audio**: Includes a built-in, toggleable background music player to enhance the formal and emotional atmosphere of the invitation.

## Technology Stack

- **Frontend:** [Next.js 14/15](https://nextjs.org/) (App Router), React, TypeScript
- **Styling:** Custom High Performance CSS Modules & CSS Variables (Vanilla CSS)
- **Backend & Database:** [Supabase](https://supabase.com) (PostgreSQL)
- **Icons & UI:** Lucide React, `canvas-confetti`
- **Deployment:** Vercel

## Project Objective

This project was developed as a comprehensive solution to modernize traditional event invitations. It demonstrates full stack capabilities, focusing on delivering a highly engaging frontend user experience rich with micro-interactions and animations while maintaining a robust, real time backend data pipeline for efficient event management.
